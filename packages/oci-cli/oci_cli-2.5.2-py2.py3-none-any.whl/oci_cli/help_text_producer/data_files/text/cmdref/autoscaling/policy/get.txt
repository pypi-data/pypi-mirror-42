get
***


Description
===========

Get Policy from a specific AutoScalingConfiguration


Usage
=====

   oci autoscaling policy get [OPTIONS]


Options
=======


--auto-scaling-configuration-id [text]
--------------------------------------

The OCID of the auto scaling configuration. [required]


--auto-scaling-policy-id [text]
-------------------------------

The ID of the auto scaling configuration policy. [required]


--from-json [text]
------------------

Provide input to this command as a JSON document from a file.

Options can still be provided on the command line. If an option exists
in both the JSON document and the command line then the command line
specified value will be used


-?, -h, --help
--------------

For detailed help on any of these individual commands, enter <command>
--help.
