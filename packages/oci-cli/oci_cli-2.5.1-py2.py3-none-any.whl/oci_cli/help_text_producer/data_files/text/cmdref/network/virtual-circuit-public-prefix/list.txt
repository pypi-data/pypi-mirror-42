list
****


Description
===========

Lists the public IP prefixes and their details for the specified
public virtual circuit.


Usage
=====

   oci network virtual-circuit-public-prefix list [OPTIONS]


Options
=======


--virtual-circuit-id [text]
---------------------------

The OCID of the virtual circuit. [required]


--verification-state [IN_PROGRESS|COMPLETED|FAILED]
---------------------------------------------------

A filter to only return resources that match the given verification
state. The state value is case-insensitive.


--all
-----

Fetches all pages of results.


--from-json [text]
------------------

Provide input to this command as a JSON document from a file.

Options can still be provided on the command line. If an option exists
in both the JSON document and the command line then the command line
specified value will be used


-?, -h, --help
--------------

For detailed help on any of these individual commands, enter <command>
--help.
