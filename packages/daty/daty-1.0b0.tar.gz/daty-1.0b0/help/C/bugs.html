Found a bug?

If you suspect you've found a bug in Daty, please get in touch about it so it can be fixed.

To help diagnose the problem as fast as possible, please include the following information:

  • Daty version and installation method (Package? Flathub? Source code?)

  • Your desktop (GNOME? KDE? Windows? Mac?)

  • Your operating system and version (Ubuntu 16.04? Fedora 28? Windows Vista? Rolled your own?)

  • Steps to reproduce the bug

  • What happened?

  • What did you expect to happen?

Thanks for your help!
