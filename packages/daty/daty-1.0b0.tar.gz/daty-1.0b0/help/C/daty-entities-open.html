Open an entity or a set of entities

To open one or more entity you have to use the open entities dialog, which is opened by default when starting the program.

To open the dialog from the editor, click on the Open button

To search for an entity by its label, type it in the search entry, then click on the result to open it in the editor.

Open multiple entities

To select more than one entity, click on the Select button then pin the checks of the entities you want to open.

To select or deselect all the entities displayed, press on the central headerbar.

You can select the row of an entity clicking on the row or by pressing the Space button when the row is selected.
