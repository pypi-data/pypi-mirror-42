BaruwaAPI
=========

Python bindings for the Baruwa REST API
---------------------------------------

|Build Status| |codecov.io| |MPLv2 License|

Installation
------------

Install from PyPi

::

    pip install BaruwaAPI

Install from Githib

::

    git clone https://github.com/baruwa-enterprise/BaruwaAPI.git
    cd BaruwaAPI
    python setup.py install
